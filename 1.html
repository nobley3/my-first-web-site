Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive THML documents from a web server of from lacal storage and render them into multimadia web pages. HTML describes the structre of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTMLpages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendred page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraghs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
